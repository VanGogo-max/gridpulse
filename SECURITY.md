# 🔒 Безопасност

## ⚠️ Важно
- **Никога не споделяйте `config.py`** — той съдържа вашите лични API ключове.
- `.gitignore` вече защитава този файл, но проверявайте преди `git commit`.

## 🔑 Препоръки за API ключове
1. Използвайте **само "трейдинг" права** (без теглене/преводи).
2. Активирайте **2FA** на борсата.
3. Ако ключът е компрометиран — **немедленно го изтрийте** от борсата.

## 🛡️ Допълнителна защита
- Работете в **изолирана среда** (отделен телефон или виртуална машина).
- Не използвайте бота с големи суми при първоначално тестване.
